Добро пожаловать в репозиторий курса по изучению **Git** и ведению базы знаний в **Obsidian**. Этот материал поможет вам освоить инструменты для управления кодом и организации личных знаний.

## 📚 О курсе

Этот репозиторий содержит материалы, лекции и практические задания, посвященные:
* Основам работы с системой контроля версий **Git**.
* Интеграции Git с редактором **Obsidian** для синхронизации и бэкапа заметок.
* Лучшим практикам организации личного Knowledge Base (базы знаний).

## 🚀 Быстрый старт

Если вы хотите сразу начать использовать Obsidian с Git, выполните следующие шаги:

### Требования
* Установленный [Git](https://git-scm.com/)
* Установленный [Obsidian](https://obsidian.md/)

## 🛠 Используемые инструменты

* **Git** — система контроля версий.
* **GitHub** — хостинг репозиториев.
* **Obsidian** — приложение для ведения заметок и построения базы знаний.

## 📝 Как пользоваться этим репозиторием

Вы можете клонировать этот репозиторий к себе на компьютер, чтобы иметь доступ к материалам курса оффлайн, или форкнуть его, если хотите вести собственные заметки поверх этого курса.

```bash
git clone https://github.com/fatalcloud/git-obsidian-course.git
```

## 🤝 Контрибьюция

Если вы нашли ошибку в материалах или хотите дополнить курс, пожалуйста, создайте Pull Request.

1. Сделайте Fork проекта.
2. Создайте ветку для ваших изменений (`git checkout -b feature/AmazingCorrection`).
3. Закоммитьте изменения (`git commit -m 'Add some AmazingCorrection'`).
4. Отправьте в ветку (`git push origin feature/AmazingCorrection`).
5. Откройте Pull Request.
