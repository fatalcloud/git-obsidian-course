Welcome to the repository for the course on learning **Git** and maintaining a knowledge base in **Obsidian**. This material will help you master the tools for code management and organizing your personal knowledge.

## 📚 About the Course

This repository contains materials, lectures, and practical assignments dedicated to:
*   The basics of working with the **Git** version control system.
*   Integrating Git with the **Obsidian** editor for syncing and backing up notes.
*   Best practices for organizing a personal Knowledge Base.

## 🚀 Quick Start

If you want to start using Obsidian with Git immediately, follow these steps:

### Requirements
*   Installed [Git](https://git-scm.com/)
*   Installed [Obsidian](https://obsidian.md/)

## 🛠 Tools Used

*   **Git** — version control system.
*   **GitHub** — repository hosting.
*   **Obsidian** — note-taking and knowledge base application.

## 📝 How to use this repository

You can clone this repository to your computer to have offline access to the course materials, or fork it if you want to keep your own notes on top of this course.

```bash
git clone https://github.com/fatalcloud/git-obsidian-course.git
```
